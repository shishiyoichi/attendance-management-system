▼アプリケーションURL
(http://localhost:8081/)

▼概要
 - JavaとSpring Bootで作成した、シンプルで直感的に操作できる勤怠管理システムです。
 - Webブラウザから手軽に出退勤を記録し、勤務時間の集計や履歴の管理を行うことができます。

▼使用技術
 - 言語: Java 21
 - フレームワーク: Spring Boot
 - テンプレートエンジン: Thymeleaf
 - フロントエンド: Bootstrap, HTML5, CSS3

▼実装機能
 - 出勤登録：現在時刻を基に出勤時間を記録
 - 退勤登録：退勤時間を記録し、当日の勤務を確定
 - 履歴一覧表示：過去のすべての出退勤記録を一覧で確認
 - 履歴編集：記録済みの勤務データを後から修正可能
 - 履歴削除：不要になった、または誤って登録した履歴を削除
 - 日別勤務時間集計：出勤から退勤までの時間を自動で計算・表示

▼工夫した点
 - 編集時はプルダウン入力を採用：自由入力による表記ゆれやフォーマットエラーを防ぎ、ユーザーが迷わずスムーズに修正できるよう、時間をプルダウンから選択するUIにしました。
 - BootstrapでUI改善：ボタンの配置や一覧表のレスポンシブ対応など、すっきりとしていて見やすく、操作しやすいクリーンなデザインを意識しました。
 - 日別勤務時間を自動計算：出勤・退勤の打刻データから、当日の稼働時間をバックエンド側で自動計算し、手計算の手間を省く仕組みを構築しました。

▼今後追加したい機能
 - MySQLの導入：現在はインメモリ/H2等のため、データの永続化を行う
 - CSV出力機能：勤務データをExcel等で管理・提出できるようにする
 - 月別集計機能：月ごとの総労働時間や稼働日数を一目で把握できるようにする
